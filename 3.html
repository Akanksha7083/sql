INSERT COMMAND

The INSERT INTO statement of SQL is used to insert a new row/record in a table. There are two ways of using the INSERT INTO statement for inserting rows:

1.INSERTING VALUES INTO ALL THE COLUMNS

The first method is to specify only the value of data to be inserted without the \ column names.
Syntax:INSERT INTO table_name VALUES (value1, value2, value3);

2.INSERTING VALUES INTO SELECTED COLUMNS

If we want to insert values in the specified columns then we use the following query syntax:
Syntax:INSERT INTO table_name(col1,col2,col3,...) VALUES(val1,val2,val3,...);
