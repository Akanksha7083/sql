AGGREGATE FUNCTIONS

In database management an aggregate function is a function where the values of multiple rows are grouped together as input on certain criteria to form a single value of more signifcant meaning.



Various Aggregate Functions

1. COUNT(column_name | *) returns the number of rows in a table.

2. SUM(column_name) returns the sum of the values of a numeric column.

3.AVG(column_name) returns the average value of a numeric column.

4. MIN(column_name) returns the minimum value of a selected column.

5.MAX(column_name) returns the maximum value of a selected column.

Syntax:SELECT group_function(col_name/*) FROM table_name;
